How To Play ?

Every Player starts with one piece with the value 3. Every player's pieces have the same color :
Player 1 : Blue
Player 2 : Red
Player 3 : Green
Player 4 : Yellow
At each player's turn, he must click on one of his pieces, by doing so, the pieces value will be incremented by 1.
When a piece reaches the value 4, it explodes. When a piece explodes it gets deleted and it increments all its neighboring pieces (up,down,left,right) and their colors change to the exploding piece's color.
For a player to win, all the pieces in the game must consume all his opponents pieces

This game is a web adaptation for the addictive android game BCG (Board Games Collection).
Developed as an open-source project by Rafaa Seddik and Ahmed Khalil Seddik.
